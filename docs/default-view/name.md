Can be used to name a view or a table.

{% highlight xml %}
    <view>
        <name>grades</name>
        <table>
            <name>students</name>
{% endhighlight %}

## View name
Gives a menu of view names for the user to select from

## Table name
This name must correspond to a name in the database.
